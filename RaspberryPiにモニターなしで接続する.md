# Raspberry Piにモニターなしで接続する
## やろうと思ったきっかけ
Raspberry Piのセットアップを行う際、マニュアルにはMicroSDの書き込み後、HDMIケーブルでモニターと接続し、GUIで設定を行うように説明があった。  
しかし、手元にモニターがなく、PCだけで設定したかったため。

## 前提
Raspberry PiをPCと同じWi-Fiに接続すること。  
Raspberry Pi ImagerでmicroSDのセットアップを行うときに設定可能。

## 試したこと
テレビにつないだ⭢だめだった笑  
自宅のWi-Fiルータの管理画面からRaspberry PiのIPアドレスを特定⭢説明書を読んだが管理画面の開き方がわからず

## 手順
1. TeraTermをインストールする。
2. TeraTermを開き、ホスト名に「[MicroSD書き込み時に設定したホスト名].local」と入力する。
3. MicroSD書き込み時に設定したユーザ名とパスワードを入力する。

ホスト名とユーザ名がわからなくなったら、MicroSD設定画面を開いて再度確認。  

## なぜ「.local」をつけると接続できるのか
mDNSという仕組みを使って名前解決されるから。  
通常の名前解決はDNSサーバに問い合わせを送信することで行われるが、 
mDNSでは、同じネットワーク内のすべての機器に名前解決の問い合わせを送信する。
同じ名前を持つ機器は自ら応答し、IPアドレスを返す。
