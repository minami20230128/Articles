# 変数のカプセル化
広い範囲で参照される変数のアクセスを関数経由にして、データの変更や追加をしやすくする。

## 例
```宣言
let defaultOwner = {firstName: "Martin", lastName: "Fowler"};
```
```参照箇所
spaceship.owner = defaultOwner;
```
```変更箇所
defaultOwner = {firstName: "Rebecca", lastName: "Persona"};
```
1. データを読み書きするアクセサを用意する。  
```
function getDefaultOwner() {return defaultOwner};
function setDefaultOwner(arg) {defaultOwner = arg};
```
2. 関数で参照をgetter、代入をsetterで置き換える。
3. 変数の可視性を変更する（privateにするなど）。
4. （データの変更まで禁止したい場合、コピーを返すようにする）
