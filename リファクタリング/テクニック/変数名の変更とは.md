# 変数名の変更とは  
変数をよりわかりやすい命名に修正すること。  

## 手順  
1. 変数が広く使われている場合は変数のカプセル化を検討
2. 変数の参照箇所をくまなく探し、すべて変更する
3. テストする

## 例1 定数名の変更  
### 前提  
ローカル変数は一つの関数でしか使われないので変更は容易。  
複数のメソッドで使われている場合は変数のカプセル化が推奨される。  
定数名の変更はこの2つと比較して慎重に行う必要がある。  

1. まずコピーする
```
const cpyNm = "Acme Gooseberries";
```

```
const companyName = "Acme Gooseberries";
const cpyNm = companyName;
```
2. 参照箇所を徐々に置き換え、テストする
3. 全部変更できたらコピーしている行を削除する
