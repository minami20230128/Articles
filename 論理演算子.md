# 論理演算子
## &と&&の違い
&&の場合、左側の条件式がfalseなら、右側の条件が何であれ全体はfalseになるので、**右側の条件式は評価されない**。
&の場合、左側の条件式が何であっても右側の条件式も評価される。（&&に比べて無駄が大きい）

以下のコードでは、a++ == 10がfalseなのでa++ == 12は評価されない。
```
class Main{
    public static void main(String[] args){
        int a = 11;
        boolean b = (a++ == 10) && (a++ == 12);
        System.out.println(a);
    }
}
```
出力結果：12

一方。以下のコードではa++ == 12となる。
```
class Main{
    public static void main(String[] args){
        int a = 11;
        boolean b = (a++ == 10) & (a++ == 12);
        System.out.println(a);
    }
}
```
出力結果：13
