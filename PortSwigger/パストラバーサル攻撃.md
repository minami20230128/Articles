# Path traversal攻撃
Path traversal攻撃とは、任意のファイル（画像など）からパスを辿ってサーバ内の別のファイルにアクセスすること。  
たとえば、Linuxサーバ上で動くシステムであれば、
```
<img src="/loadImage?filename=218.png">
```
このファイルは
```
/var/www/images/218.png
```
にあると推測できる。
そのため、その画像のURLを開き、
```
https://~filename=../../etc/passwd
```
にアクセスすればサーバのパスワードを奪取できてしまう。  

## 対策
直接サーバ内のファイル名を指定するようなコーディング方法をしない。
basename()のようなAPIを使用してパスを指定する。
